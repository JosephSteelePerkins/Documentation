SQL Database Developer

£55,000 to £65,000 / London / Feb 16, 2019

Our client is the financial consulting arm of an award-winning London-headquartered law firm.  Dynamic and professional, their highly-skilled team provide key consultancy services centred around the insurance marketplace. They are looking for a delivery-focussed SQL Database Developer to join their team.   The company expects great things from its people and, in return, provides real career development, high quality work, inspiring project teams and excellent reward and recognition programmes.  On offer is the benefit of the security of working for a large organisation together with the dynamism of a start-up environment.
 
You will be a strong SQL Database Developer, looking for the opportunity to build outstanding software and used to delivering high-quality software to tight deadlines.  Working in the reinsurance field and closely with an Actuarial team, you will be delivering bespoke solutions to complex challenges in an Agile environment.
 
With primary focus on database development and administration, plus exposure to C# and .NET, your responsibilities will include:
 
• Working closely with the product owner, actuarial and testing teams to understand the requirements, build database structures and queries
• Playing a significant role in the team, being actively involved in every step of the software development process
• Building new and modifying existing database structures and SQL queries using best practices
• Maintaining high performance in all aspects of the system, focusing on designing SQL database solutions to fulfill business requirements for scalable back-end calculation engine and web front-end
• Resolving database performance issues, using available tools and hardware configurations
 
Essential SQL Database Developer skills and experience:
 
• Microsoft SQL Server 2012 or above
• Database design including entity relationships, efficient use of keys, normalisation
• Writing performant SQL code using stored procedures and functions
• Building and maintaining data warehouses
• Implementing data migration using ETL tools and SQL scripts
• SQL Debugging, troubleshooting and optimising existing code
• Database optimisation using indexing and other performance strategies
• Git with GitFlow
• Microsoft SQL Server Management Studio
• Microsoft Visual Studio 2017
• Working as part of a collaborative team - comfortable and experienced in working in an Agile environment in a delivery-focused organisation
• Self-motivated and excellent communicator and individual, this is a great opportunity for you to
 
Desirable:

• Writing automated database unit tests
• C#, ASP.NET Web API
• Dapper
• DevOps (TeamCity, Octopus Deploy)
 
Applicants for both permanent and contract opportunities will be considered, please state your salary/rate requirement on application.
 
Please note: High levels of interest mean we will only contact you if your application is shortlisted and this will happen within five working days. You must also be eligible to work in the UK.
